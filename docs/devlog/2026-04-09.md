## Design / Decisions

- Sovereign appear_at moved to $40M (was $80M) — at $80M the player could be arrested before ever seeing the endgame's most powerful heat reducer; now functions as a strategic preparation tool
- Narrative events show one at a time, highest threshold first — prevents popup pile-up on loaded saves that crossed multiple thresholds offline
- Audio loading is fully null-safe — missing OGGs are silent no-ops, not errors; Justin can drop files in at any time without code changes
- `play_sfx` parameter renamed `sfx_name` to avoid shadowing `Node.name`
- `secret_found` signal left declared but unlistened-to — AchievementManager uses `GameState.secrets_found` counter directly (simpler, no ordering dependency)
