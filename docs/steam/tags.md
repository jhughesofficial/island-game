## Notes

- **Idle + Clicker** are the primary discovery tags — both should be present to surface in both queues.
- **Satire / Dark Comedy / Dark Humor** signals the tone and self-selects an audience comfortable with it.
- **Story Rich + Narrative + Atmospheric** justifies the game to players who want more than mechanics.
- **Political + Crime** surface it in topical searches without over-explaining.
- **Multiple Endings + Replay Value + Choices Matter** sell the Ghost Mode / prestige loop.
- **Short** sets accurate expectations — this is a session game, not a 40-hour RPG.
