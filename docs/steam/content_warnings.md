# Steam Content Warnings

## Recommended Steam Content Descriptor

**Mature Content Description (for the store page):**

> The developers describe the content like this:
>
> This game contains dark satirical themes including corruption, exploitation of power, implied criminal activity, and systemic complicity. It uses dark humor and an idle game structure to explore real-world events. There is no explicit sexual content, no graphic violence, and no gore. All activity is implied through numbers, news headlines, and narrative text.
>
> Intended for mature audiences.

## Disclosure Notes

The game depicts a fictionalized scenario clearly inspired by real events. No real individuals are named. The satirical intent is evident by the end of Act 1 and explicit by Act 3. The game does not glorify the depicted activities — the three endings include consequences ranging from arrest to a morally hollow "retirement."

The suicide-escape ending is presented as a narrative outcome, not depicted graphically. It should be noted in the content warning if Steam requires specific disclosures around that theme.
